**通过加速度（更准确地说是“加加速度/Jerk”）来控制实现“轻拿轻放”和“轻启轻停”**
描述的运动曲线，在运动控制领域称为 **“S型曲线（S-Curve）”** 或 **“七段式轨迹规划”**。它通过控制加速度的变化率（Jerk）来平滑速度曲线，从而消除梯形加减速带来的冲击和振动，是实现“轻柔”运动的**标准且最有效的方法**。



## 七段式S型曲线参数

| 段号 | 名称     | 加加速度 (Jerk) | 加速度 (Acceleration) 变化过程 | 速度 (Velocity) 变化趋势 | 实际运动表现     |
| ---- | -------- | --------------- | ------------------------------ | ------------------------ | ---------------- |
| 1    | 加加速段 | +J (正值)       | 从 0 线性增加到 +A_max         | 增加（曲率增大）         | 轻柔启动         |
| 2    | 匀加速段 | 0               | 保持 +A_max 恒定               | 线性增加                 | 快速加速         |
| 3    | 减加速段 | -J (负值)       | 从 +A_max 线性减小到 0         | 增加（曲率减小）         | 平滑过渡到匀速   |
| 4    | 恒速段   | 0               | 保持 0 恒定                    | 保持 V_max 恒定          | 高速稳定运行     |
| 5    | 减减速段 | -J (负值)       | 从 0 线性减小到 -A_max         | 减少（曲率增大）         | 开始轻柔减速     |
| 6    | 匀减速段 | 0               | 保持 -A_max 恒定               | 线性减少                 | 主要减速阶段     |
| 7    | 加减速段 | +J (正值)       | 从 -A_max 线性增加到 0         | 减少（曲率减小）         | 缓慢停止，无冲击 |



## 各阶段物理意义详解

### 1. **加加速段 (Jerk > 0)**

- **目的**：轻柔启动，避免突然加速
- **现象**：加速度从0开始缓慢增加，电机以最小的冲击开始运动
- **客户需求对应**："加速度1（正值，小）→加速度2（正值，大）"

### 2. **匀加速段 (Jerk = 0)**

- **目的**：快速建立速度
- **现象**：以最大加速度加速，速度线性快速增长
- **客户需求对应**：达到高速状态的主要加速阶段

### 3. **减加速段 (Jerk < 0)**

- **目的**：平滑过渡到匀速
- **现象**：加速度逐渐减小，速度增加变缓
- **客户需求对应**：准备进入匀速运动的缓冲阶段

### 4. **恒速段 (Jerk = 0, Acc = 0)**

- **目的**：稳定高速运行
- **现象**：以预设的最大速度V_max稳定运行
- **客户需求对应**：中间的快速运行阶段

### 5. **减减速段 (Jerk < 0)**

- **目的**：开始轻柔减速
- **现象**：减速度从0开始逐渐建立，速度开始下降
- **客户需求对应**："加速度3（负值，大）"的建立过程

### 6. **匀减速段 (Jerk = 0)**

- **目的**：快速减速
- **现象**：以最大减速度减速，速度线性快速下降
- **客户需求对应**：主要减速阶段

### 7. **加减速段 (Jerk > 0)**

- **目的**：轻柔停止，消除冲击
- **现象**：减速度逐渐减小到0，速度平滑归零
- **客户需求对应**："加速度4（负值，小）→停止"，实现"轻拿轻放"

## 实际工程应用提示

1. **Jerk值的设定**：
   - Jerk值越大，启停越"硬"，响应越快但冲击大
   - Jerk值越小，启停越"软"，运行越平稳但响应慢
   - 需要根据机械结构和负载特性现场调试找到最佳值

2. **对称性选择**：
   - 可以设置为对称曲线（启动和停止的Jerk绝对值相同）
   - 也可以设置为非对称曲线（启动和停止的Jerk值不同）
   - 晶圆搬运通常采用非对称，停止比启动更柔和

3. **轨迹生成器**：
   - 现代运动控制器都内置S型曲线轨迹生成器
   - 只需输入目标位置、速度、加速度和Jerk参数
   - 控制器会自动计算出七段式位置指令

## 当前系统分析与概念辨析

### 1. 当前PM系统设计评估

**结论**：目前的PM设计 (`pm_recipe_widget`) **无法达到** 要求的“七段式S型曲线”控制（即无法实现最佳的“轻拿轻放、轻启轻停”效果）。

主要原因有两点：
1.  **参数定义不足**：界面仅开放了“加速度”参数，而S型曲线的核心在于控制**加加速度（Jerk）**。仅设置加速度通常只能得到**梯形速度曲线（T-Curve）**，其启停瞬间加速度突变，会有刚性冲击。
2.  **代码实现缺失**：在当前的 `onStartCycle` 逻辑中，界面上配置的电机加速度参数（`motors` 数组）并没有被传递给底层的运动指令，实际运动可能仍在跑默认参数。

### 2. 伺服电机加速度 vs 物理位移加速度

| 特性 | **物理位移加速度 (Physical Acceleration)** | **伺服电机加速度 (Servo Acceleration)** |
| :--- | :--- | :--- |
| **定义** | 负载（如晶圆）在空间中实际移动的加速度。单位通常是 $m/s^2$ 或 $mm/s^2$。 | 电机转子转速变化的速率。单位通常是 $rpm/s$ (转/分/秒) 或 $counts/s^2$ (脉冲/秒²)。 |
| **关系** | **$a_{物理} = a_{电机} \times 传动比 \times 机械常数$** <br> 例如丝杆传动：$a_{线} = \alpha_{角} \times \frac{导程}{2\pi}$ | 是控制器的指令输入值。 |
| **应用** | **直接决定“轻拿轻放”的效果**。晶圆是否会滑移、震动，取决于物理加速度和加加速度。 | 工程师调试伺服驱动器时设定的参数。 |
| **差异** | 受机械刚性、皮带弹性、背隙影响，实际物理加速度可能会滞后于电机指令加速度，或产生震荡。 | 理想情况下，我们希望两者线性对应。但在S型曲线中，我们是在控制伺服加速度的**变化率**来间接控制物理加速度的平滑度。 |

### 3. 改进建议

如果要真正实现七段式S型曲线控制：

1. **底层支持**：确认 `FortrendPMCavitySubsystem` 或其下层轴卡/驱动器支持 S-Curve (Jerk) 参数设定。

2. **界面修改**：在配方中增加 **`Jerk` (加加速度)** 或 **`Smoothing Time` (平滑时间)** 参数设置，而不仅仅是加速度。

3. **代码修复**：在 `onStartCycle` 中，务必将界面配置的参数（速度、加速度、Jerk）通过 `setParameter` 或指令参数传递给底层的 `cmd` 对象

   // 伪代码示例
   auto cmdToZ2 = pmSubsystem->createLiftingActionCommand(z2);
   cmdToZ2->setAcceleration(cfg.motors[idx].lifting_axis_acce1); // 传递加速度
   cmdToZ2->setJerk(cfg.params.jerk_value); // 传递S曲线平滑度
   pmSubsystem->startCommand(cmdToZ2);
